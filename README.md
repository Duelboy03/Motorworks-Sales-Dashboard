# Motorworks-Sales-Dashboard
📌 Overview

This project presents a multi-page interactive Power BI dashboard analyzing sales performance for Motorworks. It provides insights into revenue, profitability, order trends, and product-level performance.

🛠 Tools & Technologies
	•	Power BI
	•	Power Query (ETL & Data Cleaning)
	•	DAX (Data Analysis Expressions)

📊 Key Features
	•	KPI cards for Total Revenue ($24.9M), Profit ($10.5M), Orders (25.2K), Return Rate (2.17%)
	•	Time-series analysis for revenue trends
	•	Category-wise and product-level performance breakdown
	•	Drill-down across multiple report pages

💡 Key Insights
	•	Accessories and Bikes contribute the highest revenue
	•	Certain products show high return rates affecting profitability
	•	Sales show a consistent upward trend over time
	•	Identified top-performing products driving overall sales
